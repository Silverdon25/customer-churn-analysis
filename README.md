# 📊 Customer Churn Analysis & Prediction

## 📌 Project Overview
This project analyses customer churn data to identify key factors influencing customer retention and builds a machine learning model to predict churn.

The goal is to help businesses understand why customers leave and take action to improve retention.

---

## 📂 Dataset
The dataset contains over **7,000 customer records**, including:

- Demographics (gender, senior citizen, dependents)
- Account details (tenure, contract type)
- Services (internet, phone, streaming)
- Billing (monthly and total charges)
- Target variable: **Churn**

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Seaborn & Matplotlib
- Scikit-learn

---

## 📊 Exploratory Data Analysis (EDA)
Key analysis performed:
- Churn distribution visualization
- Data cleaning and preprocessing
- Handling missing values
- Feature exploration and relationships

---

## 📈 Key Insights
- Customers with **shorter tenure** are more likely to churn
- **Month-to-month contracts** have the highest churn rate
- Higher **monthly charges** increase churn risk
- **Fiber optic customers** show higher churn behaviour

---

## 🤖 Machine Learning Model
- Model: **Random Forest Classifier**
- Accuracy: **~84%**
- Features used: tenure, contract type, monthly charges, and service-related variables

---

## 📊 Model Evaluation
The model was evaluated using:
- Accuracy
- Precision
- Recall
- F1-score

The results show strong performance in identifying customers likely to churn.

---

## 💼 Business Impact
This analysis can help businesses:
- Identify high-risk customers early
- Improve retention strategies
- Reduce revenue loss
- Make data-driven decisions
