# Customer Churn Analysis & Prediction

## 📌 Project Overview

This project analyses customer churn data to identify key factors influencing customer retention and builds a machine learning model to predict churn.

## 📊 Dataset

The dataset contains over 7,000 customer records including:

* Demographics (gender, senior citizen, dependents)
* Account details (tenure, contract type)
* Services (internet, phone, streaming)
* Billing (monthly and total charges)
* Target variable: Churn

## 🔧 Tools & Technologies

* Python
* Pandas
* Seaborn & Matplotlib
* Scikit-learn

## 📈 Key Insights

* Customers with shorter tenure are more likely to churn
* Month-to-month contracts have the highest churn rate
* Higher monthly charges increase churn risk
* Fiber optic customers show higher churn behaviour

## 🤖 Machine Learning Model

* Model: Random Forest Classifier
* Accuracy: ~80–85%
* Key features: tenure, contract type, monthly charges
